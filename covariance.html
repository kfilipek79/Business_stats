Author

Kamil Filipek

5.1.1. What Is Covariance?

Covariance measures the direction and degree to which two variables change together. If both variables tend to be above their respective means at the same time, the covariance is positive; if one tends to be high when the other is low, the covariance is negative; if they move independently, the covariance is close to zero.

\[\text{cov}(X, Y) = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{n - 1}\]

Intuitively, each term \((x_i - \bar{x})(y_i - \bar{y})\) is positive when both variables are simultaneously above or below their means, and negative otherwise.

5.1.2. Manual Calculation

# R example
x <- c(25, 27, 29, 31, 33)
y <- c(5,  15,  9, 20, 25)

x_dev <- x - mean(x)
y_dev <- y - mean(y)

cov_manual <- sum(x_dev * y_dev) / (length(x) - 1)
cat("Manual covariance: ", round(cov_manual, 2), "\n")

# Built-in
cat("cov() result:      ", round(cov(x, y), 2), "\n")
import numpy as np

x = [25, 27, 29, 31, 33]
y = [5,  15,  9, 20, 25]

# Manual
x_dev = [xi - np.mean(x) for xi in x]
y_dev = [yi - np.mean(y) for yi in y]
cov_manual = sum(xd * yd for xd, yd in zip(x_dev, y_dev)) / (len(x) - 1)
print(f"Manual covariance:  {cov_manual:.2f}")

# NumPy — returns 2×2 matrix; off-diagonal element is cov(x,y)
print(f"NumPy covariance:   {np.cov(x, y)[0, 1]:.2f}")

5.1.3. The Covariance Matrix

When analysing more than two variables, we compute a covariance matrix — a symmetric matrix where the diagonal entries are variances and the off-diagonal entries are pairwise covariances.

\[\Sigma = \begin{pmatrix} \text{var}(X_1) & \text{cov}(X_1, X_2) & \cdots \\ \text{cov}(X_2, X_1) & \text{var}(X_2) & \cdots \\ \vdots & \vdots & \ddots \end{pmatrix}\]

library(tidyverse)

set.seed(42)
df <- tibble(
  revenue    = rnorm(100, 500, 100),
  ad_spend   = rnorm(100, 50, 15),
  staff      = sample(10:50, 100, replace = TRUE),
  profit     = rnorm(100, 120, 40)
)

# Add a realistic relationship
df <- df |> mutate(revenue = revenue + 3 * ad_spend)

# Covariance matrix
cov_matrix <- cov(df)
print(round(cov_matrix, 1))

5.1.4. Limitation: Scale Dependence

The most important limitation of covariance is that it is scale-dependent.

Consider: - Revenue measured in euros vs. thousands of euros — the covariance changes by a factor of 1 000 - Comparing cov(Revenue, Headcount) = 12 000 with cov(Score, Time) = 12 000 is meaningless

This makes it impossible to answer the question “which pair of variables is more strongly related?” using covariance alone.

Solution: Standardise covariance → Pearson correlation coefficient (next chapter).

Property Covariance Correlation
Range \((-\infty, +\infty)\) \([-1, +1]\)
Units Product of original units Dimensionless
Interpretation Direction only Direction + strength
Comparable across variables No Yes

5.1.5. Covariance in Context: Portfolio Theory

Covariance plays a central role in Markowitz portfolio theory (modern portfolio theory). The risk of a portfolio is not just the sum of individual asset risks — it depends on the covariances between assets.

For a two-asset portfolio:

\[\sigma_p^2 = w_1^2 \sigma_1^2 + w_2^2 \sigma_2^2 + 2 w_1 w_2 \cdot \text{cov}(r_1, r_2)\]

Where \(w_i\) is the portfolio weight and \(\sigma_i^2\) is the variance of asset \(i\).

  • If \(\text{cov}(r_1, r_2) < 0\) — assets move in opposite directions → diversification reduces risk
  • If \(\text{cov}(r_1, r_2) > 0\) — assets move together → less diversification benefit
# Simulate two stock return series
set.seed(1)
n <- 252   # trading days

r1 <- rnorm(n, mean = 0.001, sd = 0.015)  # Asset A
r2 <- -0.5 * r1 + rnorm(n, 0, 0.012)      # Asset B (negatively correlated)

cat("Cov(A, B):", round(cov(r1, r2), 6), "\n")
cat("Portfolio variance (50/50):",
    round(0.25 * var(r1) + 0.25 * var(r2) + 2 * 0.5 * 0.5 * cov(r1, r2), 6), "\n")
Note

Key insight: Two assets with negative covariance reduce portfolio variance below the weighted average of individual variances — the mathematical foundation of diversification.
