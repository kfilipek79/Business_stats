3.3. Chi-Square Test and t-Tests

Author

Kamil Filipek

Two of the most frequently used statistical tests in business research are the chi-square test (for categorical data) and the t-test (for comparing means of continuous data).

Part A: Chi-Square Test (\(\chi^2\))

3.3.1. What Is the Chi-Square Test?

Pearson’s chi-square test is a non-parametric test for categorical variables. It tests whether observed frequencies are significantly different from expected frequencies.

There are two variants:

Test Purpose
Goodness of fit Does one categorical variable follow an expected distribution?
Test of independence Are two categorical variables related to each other?

Both use the same formula:

\[\chi^2 = \sum \frac{(O - E)^2}{E}\]

where \(O\) is the observed frequency and \(E\) is the expected frequency under H₀.

3.3.2. Chi-Square Test of Independence

Example: A retail chain wants to know whether customer gender is associated with product category preference.

H₀: Gender and product preference are independent. H₁: There is an association between gender and preference.

import numpy as np
from scipy.stats import chi2_contingency
import pandas as pd

# Observed contingency table
observed = np.array([
    [120, 80, 50],   # Female: Electronics, Clothing, Food
    [90,  40, 70],   # Male:   Electronics, Clothing, Food
])

chi2, p, dof, expected = chi2_contingency(observed)

print(f"Chi-square statistic: {chi2:.3f}")
print(f"Degrees of freedom:   {dof}")
print(f"p-value:              {p:.4f}")

if p < 0.05:
    print("Reject H0 — there IS an association between gender and product preference")
else:
    print("Fail to reject H0 — no significant association")

3.3.3. Goodness of Fit

Example: A marketing team expects customers to visit equally across five weekdays. Actual visit counts differ — is this due to chance?

from scipy.stats import chisquare

observed = [95, 112, 88, 130, 75]   # Mon–Fri visits
expected = [100, 100, 100, 100, 100]  # uniform expectation

chi2, p = chisquare(f_obs=observed, f_exp=expected)
print(f"chi2 = {chi2:.3f}, p = {p:.4f}")

3.3.4. Assumptions

  • Observations are independent
  • Each cell expected frequency ≥ 5 (use Fisher’s exact test for small samples)
  • Data must be counts (not percentages or means)

Part B: Student’s t-Test

3.3.5. What Is the t-Test?

The t-test compares the means of continuous variables and determines whether observed differences are statistically significant. It is a parametric test, assuming normality (or n large enough for CLT to apply) and independence.

Three types:

Type When to use
One-sample Compare a sample mean to a known/hypothesised value
Independent two-sample Compare means of two separate groups
Paired Compare means before and after (same subjects)

3.3.6. One-Sample t-Test

Example: The standard delivery time is 5 days. A sample of 30 deliveries yields a mean of 5.8 days. Is this significantly different?

from scipy import stats
import numpy as np

np.random.seed(1)
deliveries = np.random.normal(loc=5.8, scale=1.5, size=30)

t_stat, p_value = stats.ttest_1samp(deliveries, popmean=5.0)

print(f"Sample mean:  {deliveries.mean():.2f} days")
print(f"t = {t_stat:.3f}, p = {p_value:.4f}")

3.3.7. Independent Two-Sample t-Test

Example: Does the new website layout (Group B) produce higher average session times than the old layout (Group A)?

from scipy import stats
import numpy as np

np.random.seed(42)
group_A = np.random.normal(loc=3.2, scale=0.8, size=50)  # Old layout
group_B = np.random.normal(loc=3.7, scale=0.9, size=50)  # New layout

t_stat, p_value = stats.ttest_ind(group_A, group_B, equal_var=False)  # Welch's t-test

print(f"Mean A: {group_A.mean():.2f} min  |  Mean B: {group_B.mean():.2f} min")
print(f"t = {t_stat:.3f}, p = {p_value:.4f}")

if p_value < 0.05:
    print("Reject H0 — the new layout significantly increases session time")

3.3.8. Paired t-Test

Example: Do employees perform better after a training programme?

from scipy import stats
import numpy as np

np.random.seed(10)
before = np.random.normal(loc=65, scale=10, size=25)
after  = before + np.random.normal(loc=8, scale=5, size=25)  # improvement

t_stat, p_value = stats.ttest_rel(before, after)

print(f"Mean before: {before.mean():.1f}  |  Mean after: {after.mean():.1f}")
print(f"Mean difference: {(after - before).mean():.1f}")
print(f"t = {t_stat:.3f}, p = {p_value:.4f}")

3.3.9. Assumptions and When to Use Non-Parametric Alternatives

Assumption t-Test Violated → use
Normality (or large n) Required Wilcoxon signed-rank (paired), Mann-Whitney U (independent)
Independence Required Paired t-test if same subjects
Equal variances For standard 2-sample Use Welch’s t-test (equal_var=False)

3.3.10. One-Tailed vs. Two-Tailed Tests

  • Two-tailed (H₁: μ₁ ≠ μ₂): testing for any difference, regardless of direction
  • One-tailed (H₁: μ₁ > μ₂ or μ₁ < μ₂): testing for a difference in a specific direction

In practice, two-tailed tests are preferred unless there is a strong a priori theoretical reason to test in one direction only — one-tailed tests are easier to “pass” and are prone to misuse.

# Two-tailed (default)
t, p_two = stats.ttest_ind(group_A, group_B, equal_var=False)

# One-tailed (testing if B > A)
p_one = p_two / 2 if t < 0 else 1 - p_two / 2

print(f"Two-tailed p: {p_two:.4f}")
print(f"One-tailed p (B > A): {p_one:.4f}")
