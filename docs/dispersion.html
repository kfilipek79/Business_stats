2.2. Variability and Spread

Author

Kamil Filipek

Measures of central tendency tell us where data is centred, but two datasets can have identical means and very different distributions. Measures of variability describe how spread out the values are.

2.2.1. Range

The simplest measure — the difference between the maximum and minimum value:

\[\text{Range} = X_{\max} - X_{\min}\]

data = (3, 2, 5, 18, 11, 6, 7, 9, 12, 35, 3, 5, 6, 2, 2, 3, 9, 5, 8, 8, 21, 3)
print(f"Min: {min(data)}, Max: {max(data)}, Range: {max(data) - min(data)}")
# Min: 2, Max: 35, Range: 33

Limitation: extremely sensitive to outliers; tells nothing about the distribution in between.

2.2.2. Variance

Variance measures the average squared deviation from the mean.

Population variance: \[\sigma^2 = \frac{\sum_{i=1}^{N}(x_i - \mu)^2}{N}\]

Sample variance (Bessel’s correction — dividing by \(N-1\) makes it an unbiased estimator): \[S^2 = \frac{\sum_{i=1}^{N}(x_i - \bar{X})^2}{N - 1}\]

import statistics

data = [3, 2, 5, 18, 11, 6, 7, 9, 12, 35]
print(f"Sample variance: {statistics.variance(data):.2f}")

import numpy as np
print(f"Population variance: {np.var(data):.2f}")
print(f"Sample variance:     {np.var(data, ddof=1):.2f}")

2.2.3. Standard Deviation

Standard deviation is the square root of variance, bringing the measure back to the original units:

\[\sigma = \sqrt{\sigma^2} \qquad S = \sqrt{S^2}\]

import numpy as np

data = [3, 2, 5, 18, 11, 6, 7, 9, 12, 35]
print(f"Standard deviation: {np.std(data, ddof=1):.2f}")

Interpretation rule of thumb (normal distribution): - ~68% of values fall within ±1 SD of the mean - ~95% within ±2 SD - ~99.7% within ±3 SD

2.2.4. Coefficient of Variation

Standard deviation depends on the scale of the variable — it is not directly comparable across datasets measured in different units. The coefficient of variation (CV) normalises SD by the mean:

\[CV = \frac{S}{\bar{X}} \times 100\%\]

import numpy as np

prices_A = [100, 102, 98, 105, 97]   # Product A (stable)
prices_B = [20,  35,  15, 40,  10]   # Product B (volatile)

for name, series in [("A", prices_A), ("B", prices_B)]:
    cv = np.std(series, ddof=1) / np.mean(series) * 100
    print(f"Product {name} — SD: {np.std(series, ddof=1):.2f}, CV: {cv:.1f}%")

2.2.5. Skewness

Skewness measures the asymmetry of a distribution.

\[\text{Skewness} = \frac{3(\bar{X} - M_{\text{median}})}{S}\]

  • Positive skew (right-skewed): long tail to the right; mean > median. Common in income, wealth, wait times.
  • Negative skew (left-skewed): long tail to the left; mean < median.
  • Zero skew: symmetric distribution (e.g., normal distribution).
from scipy.stats import skew
import matplotlib.pyplot as plt

data = [1, 2, 2, 3, 3, 3, 4, 5, 8, 15, 30]
print(f"Skewness: {skew(data):.3f}")

plt.hist(data, bins=8, color="steelblue", edgecolor="white")
plt.title(f"Right-skewed distribution (skewness = {skew(data):.2f})")
plt.xlabel("Value")
plt.show()

2.2.6. Kurtosis

Kurtosis describes the “weight” or thickness of the tails relative to a normal distribution.

\[\text{Kurtosis} = \frac{\sum(x_i - \bar{X})^4}{\sigma^4 \cdot N}\]

Type Kurtosis value Tail weight
Mesokurtic (normal) ≈ 3 Normal tails
Leptokurtic > 3 Fat tails (extreme values more likely)
Platykurtic < 3 Thin tails (extreme values less likely)

In finance, fat tails are crucial: the Black-Scholes model assumes normal returns, but real markets are leptokurtic — extreme events (“black swans”) are far more common than the model predicts.

from scipy.stats import kurtosis

data = (3, 2, 5, 18, 11, 6, 7, 9, 12, 35, 3, 5, 6, 2, 2, 3, 9, 5, 8, 8, 21, 3)
print(f"Kurtosis (excess): {kurtosis(data):.3f}")
# scipy returns excess kurtosis (normal = 0); add 3 to get the raw value

2.2.7. Summary Statistics in One Line

import pandas as pd

data = pd.Series([3, 2, 5, 18, 11, 6, 7, 9, 12, 35, 3, 5, 6, 2, 2, 3, 9, 5, 8, 8, 21, 3])

print(data.describe())
# count, mean, std, min, 25%, 50%, 75%, max

The describe() output gives a five-number summary (min, Q1, median, Q3, max) plus mean and standard deviation — the foundation of any exploratory data analysis.

2.2.8. Box Plot — Visualising Variability

A box plot (box-and-whisker plot) displays the five-number summary graphically and highlights outliers.

import matplotlib.pyplot as plt
import pandas as pd

# Monthly revenue for three product lines (€k)
data = {
    "Electronics":  [82, 91, 88, 79, 200, 85, 90, 93],
    "Clothing":     [45, 48, 50, 47, 43,  46, 49, 51],
    "Food":         [30, 31, 29, 35, 28,  33, 32, 30],
}
df = pd.DataFrame(data)

df.boxplot(figsize=(8, 5))
plt.ylabel("Revenue (€k)")
plt.title("Revenue distribution by product line")
plt.tight_layout()
plt.show()
