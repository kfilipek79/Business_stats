5.2. Structural Equation Modelling

Author

Kamil Filipek

5.2.1. What Is SEM?

Structural Equation Modelling (SEM) is a multivariate statistical framework that simultaneously estimates:

  1. A measurement model — how observed indicators (survey items, tests) relate to latent (unobservable) constructs (e.g., “job satisfaction”, “brand loyalty”)
  2. A structural model — causal and correlational relationships among the constructs

SEM extends multiple regression in three important ways:

Feature Multiple regression SEM
Measurement error Assumed = 0 Explicitly modelled
Latent variables Not possible Core feature
Multiple outcomes One at a time All simultaneously
Indirect effects Require extra steps Built-in

5.2.2. Key Components

Observed variables (manifest) — variables directly measured (e.g., survey items Q1–Q5)

Latent variables (constructs) — theoretical concepts estimated from indicators: - Exogenous (ξ, “xi”) — purely independent, not caused by any other construct in the model - Endogenous (η, “eta”) — dependent on at least one other construct

Paths: - Factor loadings (λ) — relationship between an indicator and its latent construct - Structural paths (β, γ) — relationships between latent constructs - Residuals/errors (ε, ζ) — unexplained variance

5.2.3. A Conceptual Example

Consider an HR study of the relationship between Job Demands, Resources, and Burnout:

Job Demands (ξ₁) ──────────────────── → Burnout (η₂)
                                          ↑
Resources (ξ₂) → Work Engagement (η₁) ──┘
  • Job Demands and Resources are measured by survey items (observed)
  • Work Engagement and Burnout are latent constructs
  • We test whether Resources reduce Burnout indirectly through Work Engagement

5.2.4. Confirmatory Factor Analysis (CFA)

CFA is the measurement part of SEM — it tests whether observed indicators reliably measure a latent construct.

# R example using lavaan
library(lavaan)

model_cfa <- '
  # Latent variable definitions
  satisfaction =~ q1 + q2 + q3 + q4
  loyalty      =~ q5 + q6 + q7
'

fit_cfa <- cfa(model_cfa, data = my_data, estimator = "MLR")
summary(fit_cfa, fit.measures = TRUE, standardized = TRUE)

Key CFA output: - Factor loadings (standardised): should be > 0.50 (ideally > 0.70) - Fit indices: CFI > 0.95, RMSEA < 0.06, SRMR < 0.08 indicate good fit - Reliability: Cronbach’s α > 0.70; composite reliability (CR) > 0.70

5.2.5. Full SEM in R (lavaan)

library(lavaan)

# Model specification
model_sem <- '
  # Measurement model
  job_demands  =~ jd1 + jd2 + jd3
  resources    =~ res1 + res2 + res3 + res4
  engagement   =~ eng1 + eng2 + eng3
  burnout      =~ bo1 + bo2 + bo3

  # Structural model (causal paths)
  engagement ~ resources + job_demands
  burnout    ~ job_demands + engagement

  # Indirect effect (mediation)
  indirect := resources * engagement
'

fit <- sem(model_sem, data = hr_data, estimator = "MLR")

# Summary with fit statistics and standardised estimates
summary(fit,
        fit.measures = TRUE,
        standardized = TRUE,
        rsquare      = TRUE)

5.2.6. Model Fit Assessment

A SEM result without fit evaluation is meaningless. Report:

Index Good fit Acceptable
CFI (Comparative Fit Index) ≥ 0.95 ≥ 0.90
TLI (Tucker-Lewis Index) ≥ 0.95 ≥ 0.90
RMSEA ≤ 0.06 ≤ 0.08
SRMR ≤ 0.08 ≤ 0.10
χ²/df ≤ 2 ≤ 3–5
Warning

Never use χ² alone as a fit index — it is extremely sensitive to sample size and almost always significant for n > 200.

5.2.7. Mediation Analysis

SEM makes mediation analysis straightforward. An indirect effect exists when X affects Y through an intervening variable M:

\[X \to M \to Y\]

Total effect = Direct effect + Indirect effect: \[c = c' + a \cdot b\]

# Bootstrap confidence interval for indirect effect (gold standard)
fit_boot <- sem(model_sem, data = hr_data,
                estimator = "MLR",
                se = "bootstrap",
                bootstrap = 1000)

# Extract indirect effect with CI
parameterEstimates(fit_boot,
                   boot.ci.type = "perc",
                   level = 0.95)[parameterEstimates(fit_boot)$label == "indirect", ]

5.2.8. When to Use SEM

SEM is appropriate when:

  • Variables of interest cannot be directly measured (attitudes, intelligence, brand equity)
  • You need to test a complete theoretical model rather than isolated bivariate relationships
  • You want to account for measurement error explicitly
  • You have multiple dependent variables that are causally related to each other

SEM requires larger samples than regression — a common rule of thumb is at least 200 observations, with 10–20 observations per estimated parameter.
