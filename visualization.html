3.2. Data Visualization

Author

Kamil Filipek

3.2.1. Why Visualisation Matters

A well-chosen plot communicates in seconds what a table of numbers takes minutes to interpret. Visualisation is not decoration — it is a core analytical tool:

  • Exploratory data analysis (EDA): spot distributions, outliers, and patterns before modelling
  • Communication: present results to non-technical stakeholders
  • Diagnosis: check model assumptions (residuals, Q-Q plots)
  • Quality control: identify data entry errors and anomalies

“The greatest value of a picture is when it forces us to notice what we never expected to see.” — John Tukey

3.2.2. Choosing the Right Chart

Goal Chart type
Distribution of one variable Histogram, density plot, box plot
Comparison across categories Bar chart, dot plot
Relationship between two numeric variables Scatter plot
Change over time Line chart
Part-of-whole Pie chart (use sparingly), stacked bar
Many variables at once Heatmap, scatter matrix, parallel coordinates

Rule of thumb: if your chart requires a legend to decode more than 3–4 categories, consider splitting into facets.

3.2.3. Histograms and Density Plots

Histograms show the distribution of a numeric variable by counting observations in equal-width bins.

library(ggplot2)

set.seed(42)
df <- data.frame(income = rnorm(500, mean = 45000, sd = 12000))

# Histogram
ggplot(df, aes(x = income)) +
  geom_histogram(bins = 30, fill = "steelblue", colour = "white") +
  labs(title = "Income distribution",
       x = "Annual income (€)", y = "Count") +
  theme_minimal()

# Histogram + density curve
ggplot(df, aes(x = income)) +
  geom_histogram(aes(y = after_stat(density)),
                 bins = 30, fill = "steelblue", colour = "white", alpha = 0.6) +
  geom_density(colour = "firebrick", linewidth = 1) +
  labs(title = "Income — histogram with density", x = "Annual income (€)") +
  theme_minimal()
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

np.random.seed(42)
income = np.random.normal(45000, 12000, 500)

fig, axes = plt.subplots(1, 2, figsize=(12, 4))

axes[0].hist(income, bins=30, color="steelblue", edgecolor="white")
axes[0].set(title="Histogram", xlabel="Annual income (€)", ylabel="Count")

sns.histplot(income, kde=True, ax=axes[1], color="steelblue")
axes[1].set(title="Histogram + KDE", xlabel="Annual income (€)")

plt.tight_layout()
plt.show()

3.2.4. Box Plots

Box plots summarise the five-number summary (min, Q1, median, Q3, max) and highlight outliers.

library(ggplot2)
library(dplyr)

set.seed(1)
df <- data.frame(
  department = rep(c("Sales", "HR", "IT", "Finance"), each = 60),
  salary     = c(rnorm(60, 55000, 8000),
                 rnorm(60, 48000, 6000),
                 rnorm(60, 72000, 12000),
                 rnorm(60, 65000, 10000))
)

ggplot(df, aes(x = reorder(department, salary, median), y = salary, fill = department)) +
  geom_boxplot(show.legend = FALSE, outlier.colour = "red", outlier.shape = 1) +
  coord_flip() +
  labs(title = "Salary distribution by department",
       x = NULL, y = "Annual salary (€)") +
  scale_y_continuous(labels = scales::comma) +
  theme_minimal()
import seaborn as sns
import matplotlib.pyplot as plt

departments = ["Sales"] * 60 + ["HR"] * 60 + ["IT"] * 60 + ["Finance"] * 60
salaries = (
    list(np.random.normal(55000, 8000, 60)) +
    list(np.random.normal(48000, 6000, 60)) +
    list(np.random.normal(72000, 12000, 60)) +
    list(np.random.normal(65000, 10000, 60))
)

import pandas as pd
df = pd.DataFrame({"Department": departments, "Salary": salaries})

plt.figure(figsize=(8, 5))
order = df.groupby("Department")["Salary"].median().sort_values().index
sns.boxplot(data=df, x="Salary", y="Department", order=order, palette="Set2")
plt.title("Salary distribution by department")
plt.tight_layout()
plt.show()

3.2.5. Bar Charts

Use bar charts for categorical comparisons of counts or aggregated values.

library(ggplot2)
library(dplyr)

sales <- data.frame(
  region  = c("North", "South", "East", "West", "Central"),
  revenue = c(420, 380, 510, 290, 460)
)

ggplot(sales, aes(x = reorder(region, -revenue), y = revenue, fill = region)) +
  geom_col(show.legend = FALSE) +
  geom_text(aes(label = paste0("€", revenue, "k")),
            vjust = -0.4, size = 3.5) +
  labs(title = "Revenue by region (2023)", x = NULL, y = "Revenue (€000s)") +
  theme_minimal()
Tip

Always order bars by value (unless the categories have an inherent order). A random ordering forces the reader’s eye to hunt for patterns.

3.2.6. Scatter Plots

Scatter plots reveal the relationship between two numeric variables.

library(ggplot2)

set.seed(7)
n <- 150
df <- data.frame(
  ad_spend = runif(n, 10, 100),
  revenue  = 200 + 3.5 * runif(n, 10, 100) + rnorm(n, 0, 30),
  segment  = sample(c("SME", "Enterprise", "Startup"), n, replace = TRUE)
)

ggplot(df, aes(x = ad_spend, y = revenue, colour = segment)) +
  geom_point(alpha = 0.7, size = 2.5) +
  geom_smooth(method = "lm", se = TRUE, linewidth = 0.8) +
  labs(title = "Ad spend vs. Revenue by segment",
       x = "Ad spend (€000s)", y = "Revenue (€000s)",
       colour = "Segment") +
  theme_minimal()
import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

np.random.seed(7)
n = 150
df = pd.DataFrame({
    "ad_spend": np.random.uniform(10, 100, n),
    "revenue":  200 + 3.5 * np.random.uniform(10, 100, n) + np.random.normal(0, 30, n),
    "segment":  np.random.choice(["SME", "Enterprise", "Startup"], n)
})

plt.figure(figsize=(8, 5))
sns.scatterplot(data=df, x="ad_spend", y="revenue",
                hue="segment", alpha=0.7)
sns.regplot(data=df, x="ad_spend", y="revenue",
            scatter=False, color="grey", line_kws={"linewidth": 1.5})
plt.title("Ad spend vs. Revenue")
plt.tight_layout()
plt.show()

3.2.7. Line Charts (Time Series)

library(ggplot2)
library(dplyr)
library(lubridate)

set.seed(3)
ts_data <- data.frame(
  date    = seq(as.Date("2022-01-01"), as.Date("2023-12-01"), by = "month"),
  revenue = cumsum(rnorm(24, mean = 5, sd = 3)) + 100
)

ggplot(ts_data, aes(x = date, y = revenue)) +
  geom_line(colour = "steelblue", linewidth = 1) +
  geom_point(colour = "steelblue", size = 2) +
  labs(title = "Monthly revenue 2022–2023",
       x = NULL, y = "Revenue (€000s)") +
  scale_x_date(date_labels = "%b %Y", date_breaks = "3 months") +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 30, hjust = 1))

3.2.8. Faceting — Small Multiples

When comparing multiple subgroups, facets (small multiples) are often cleaner than overlaying on one plot.

library(ggplot2)

set.seed(5)
df <- data.frame(
  year   = rep(2019:2023, each = 50),
  profit = c(rnorm(50, 100, 20), rnorm(50, 105, 22),
             rnorm(50, 98,  25), rnorm(50, 115, 18),
             rnorm(50, 130, 20))
)

ggplot(df, aes(x = profit)) +
  geom_histogram(bins = 20, fill = "steelblue", colour = "white") +
  facet_wrap(~year, nrow = 1) +
  labs(title = "Profit distribution by year", x = "Profit (€000s)", y = "Count") +
  theme_minimal()

3.2.9. ggplot2 Grammar of Graphics

Every ggplot2 plot is built from layers:

ggplot(data = <DATA>,              # 1. data frame
       aes(x = <VAR>, y = <VAR>,   # 2. aesthetics: what maps to what
           colour = <VAR>,
           fill   = <VAR>)) +
  geom_<TYPE>() +                  # 3. geometric object (bar, point, line, ...)
  stat_<FUN>() +                   # 4. statistical transformation (optional)
  scale_<AES>_<TYPE>() +           # 5. axis/colour scales
  facet_wrap(~<VAR>) +             # 6. faceting (optional)
  labs(title = "...",              # 7. labels
       x = "...", y = "...") +
  theme_minimal()                  # 8. theme (appearance)

Common geoms:

Geom Chart type
geom_point() Scatter plot
geom_line() Line chart
geom_col() / geom_bar() Bar chart
geom_histogram() Histogram
geom_density() Density curve
geom_boxplot() Box plot
geom_violin() Violin plot
geom_tile() Heatmap
geom_smooth() Trend line (with CI)

3.2.10. Visualisation Best Practices

Do: - Use clear, descriptive titles and axis labels with units - Order categorical axes meaningfully (by value, not alphabetically) - Use colourblind-friendly palettes (viridis, ColorBrewer) - Show uncertainty (confidence intervals, error bars) - Keep it simple — remove chart junk

Avoid: - 3D charts (they distort perception of area) - Pie charts with more than 4 slices - Dual y-axes (they mislead about relationships) - Truncated y-axes that exaggerate differences - Colour for decoration rather than information

# Colourblind-friendly palette
library(ggplot2)

ggplot(df, aes(x = department, y = salary, fill = department)) +
  geom_col(show.legend = FALSE) +
  scale_fill_brewer(palette = "Set2") +   # or "viridis"
  theme_minimal()
